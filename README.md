# Kupunya CV

Aplikasi Android sederhana bertema biodata/CV yang dibuat menggunakan **Jetpack Compose**.  
Project ini merupakan tugas untuk menampilkan informasi biodata pribadi seperti **foto, nama, NIM, riwayat pendidikan, pengalaman, skill, dan bahasa** dengan tampilan UI yang modern.

## Fitur
- Menampilkan foto profil
- Menampilkan nama, NIM, dan kelas
- Menampilkan deskripsi singkat / About Me
- Menampilkan riwayat pendidikan
- Menampilkan pengalaman dan pencapaian
- Menampilkan skill dan tools
- Menampilkan kemampuan bahasa
- Menggunakan tampilan modern dengan aksen warna ungu kebiruan

## Teknologi yang Digunakan
- Kotlin
- Android Studio
- Jetpack Compose
- Material 3

## Cara Menjalankan Project

1. Clone repository ini
2. Buka project di **Android Studio**
3. Tunggu proses **Gradle Sync** selesai
4. Pastikan file foto tersedia di folder:
   `app/src/main/res/drawable/`
5. Jalankan aplikasi di emulator atau perangkat Android
